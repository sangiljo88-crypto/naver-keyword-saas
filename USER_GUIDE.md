4. 구매 확인

⚠️ **주의**: 현재는 간단한 구독 모델만 구현되어 있습니다. 실제 결제는 연동되지 않았습니다.

### 3. API 키 설정

키워드 분석을 위해서는 네이버 API 키가 필요합니다.

#### 네이버 광고 API 키 발급

1. [네이버 광고 관리 시스템](https://manage.searchad.naver.com/) 접속
2. 로그인 후 "도구" > "API 관리" 메뉴 선택
3. API 키 발급
4. 다음 정보 확인:
   - ACC KEY
   - SECRET KEY
   - CUSTOMER_ID

#### 네이버 개발자센터 API 키 발급

1. [네이버 개발자센터](https://developers.naver.com/) 접속
2. "Application" > "애플리케이션 등록" 클릭
3. 애플리케이션 정보 입력
4. "검색" API 선택
5. 다음 정보 확인:
   - Client ID
   - Client Secret

#### API 키 저장

1. 좌측 사이드바에서 "API 설정" 클릭
2. 네이버 광고 API 정보 입력:
   - ACC KEY
   - SECRET KEY (비밀키)
   - CUSTOMER_ID
3. 네이버 개발자센터 API 정보 입력:
   - Client ID
   - Client Secret
4. "저장" 버튼 클릭

## 📊 키워드 분석 기능

### 1. 블로그 키워드 추출

**용도**: 블로그 글 작성을 위한 키워드 검색량 및 경쟁 정도 분석

**사용 방법**:
1. 좌측 사이드바에서 "블로그 키워드 추출" 클릭
2. 키워드 입력란에 분석할 키워드 입력
3. "키워드 체크" 버튼 클릭
4. 결과 확인:
   - **PC**: PC 검색량
   - **MO**: 모바일 검색량
   - **SUM**: 합계 검색량
   - **문서 수**: 블로그 문서 수
   - **비율**: 경쟁 비율 (문서수/검색량)

**활용 팁**:
- 경쟁 비율이 낮을수록 진입하기 쉬운 키워드
- 검색량이 높고 문서 수가 적은 키워드가 좋은 키워드

### 2. 상품 키워드 추출

**용도**: 쇼핑몰 상품 등록을 위한 키워드 검색량 및 경쟁 정도 분석

**사용 방법**:
1. 좌측 사이드바에서 "상품 키워드 추출" 클릭
2. 키워드 입력란에 분석할 키워드 입력
3. "키워드 체크" 버튼 클릭
4. 결과 확인:
   - **PC**: PC 검색량
   - **MO**: 모바일 검색량
   - **SUM**: 합계 검색량
   - **상품 수**: 네이버 쇼핑 상품 수
   - **비율**: 경쟁 비율 (상품수/검색량)

**활용 팁**:
- 상품 수가 적고 검색량이 많은 키워드 선택
- 모바일 검색량이 높은 키워드가 유리

### 3. 빠른 검색량 조회

**용도**: 빠르게 키워드 검색량만 확인

**사용 방법**:
1. 좌측 사이드바에서 "빠른 검색량 조회" 클릭
2. 키워드 입력
3. 결과 확인:
   - PC 검색량
   - 모바일 검색량
   - 검색 총합
   - 파워링크 수

**활용 팁**:
- 파워링크 수가 많으면 경쟁이 치열한 키워드

### 4. 대량 키워드 조회

**용도**: 많은 키워드를 한번에 조회

**사용 방법**:
1. 좌측 사이드바에서 "대량 키워드 조회" 클릭
2. 텍스트 영역에 키워드를 한 줄에 하나씩 입력
   ```
   키워드1
   키워드2
   키워드3
   ```
3. 조회 유형 선택:
   - 문서 수 조회 (블로그)
   - 상품 수 조회 (쇼핑)
4. "키워드 체크" 버튼 클릭
5. 결과 확인

**활용 팁**:
- 엑셀에서 키워드 리스트 복사 후 붙여넣기
- 최대 100개까지 한번에 조회 가능

## 📥 데이터 내보내기

### Excel/CSV 다운로드

1. 키워드 분석 후 결과 테이블 하단의 "엑셀 다운로드" 버튼 클릭
2. CSV 파일 다운로드
3. 엑셀에서 열기

```csv
키워드,PC,모바일,합계,문서수,상품수,경쟁비율
키워드1,1000,2000,3000,5000,1000,1.67
키워드2,500,1500,2000,3000,800,1.50
```

## 👤 계정 관리

### 비밀번호 변경

1. 좌측 사이드바에서 "내 정보" 클릭
2. "비밀번호 변경" 버튼 클릭
3. 현재 비밀번호 입력
4. 새 비밀번호 입력 (최소 6자)
5. "변경하기" 버튼 클릭

### 회원 탈퇴

1. 좌측 사이드바에서 "내 정보" 클릭
2. "회원탈퇴" 버튼 클릭
3. 비밀번호 입력
4. "탈퇴하기" 버튼 클릭

⚠️ **주의**: 회원 탈퇴 시 모든 데이터가 삭제됩니다.

## 🔧 API 직접 호출 (개발자용)

### 인증

```bash
# 로그인
curl -X POST https://your-domain/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"password123"}'

# 응답
{
  "token": "eyJhbGc...",
  "user": {
    "id": 1,
    "email": "user@example.com",
    "name": "홍길동"
  }
}
```

### 키워드 분석

```bash
curl -X POST https://your-domain/api/keyword/search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "keywords": ["키워드1", "키워드2"],
    "searchType": "blog"
  }'
```

### 사용량 조회

```bash
curl https://your-domain/api/keyword/usage \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## ❓ 자주 묻는 질문 (FAQ)

### Q1. API 키가 없으면 사용할 수 없나요?
A: 네, 키워드 분석을 위해서는 네이버 API 키가 필수입니다.

### Q2. 일일 사용 한도를 초과하면 어떻게 되나요?
A: 다음 날 자정에 초기화될 때까지 키워드 분석이 제한됩니다.

### Q3. 이용권이 만료되면 어떻게 되나요?
A: 키워드 분석 기능을 사용할 수 없으며, 이용권을 재구매해야 합니다.

### Q4. 여러 개의 API 키를 등록할 수 있나요?
A: 계정당 하나의 API 키 세트만 등록 가능합니다.

### Q5. 검색 이력은 얼마나 보관되나요?
A: 무제한으로 보관됩니다.

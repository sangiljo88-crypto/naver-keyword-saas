# 키워드 분석 화면 데모

## 🎨 완성된 UI

### 화면 구성

**메인 키워드 분석 화면** - 단일 페이지로 완성

#### 1. 헤더
- 프로젝트 로고 및 타이틀
- **실시간 사용량 표시**
  - 숫자 표시: 현재/제한
  - 프로그레스 바 (색상: 녹색→노랑→빨강)

#### 2. 검색 타입 탭
- 🟢 블로그 키워드 (녹색)
- 🟣 상품 키워드 (보라색)
- 🔵 빠른 검색량 (파란색)
- 🔴 대량 조회 (빨간색)

#### 3. 키워드 입력
- 텍스트 영역 (6줄)
- 실시간 키워드 개수 표시
- 플레이스홀더 예시

#### 4. 액션 버튼
- **키워드 분석** (메인 버튼, 전체 너비)
- **엑셀 다운로드** (결과 있을 때만 표시)

#### 5. 결과 테이블
- **판단 컬럼 (강조)**
  - 🔥 우수 (빨간색 배경)
  - ✅ 양호 (녹색 배경)
  - ⚠️ 경고 (노란색 배경)
  - ❌ 위험 (회색 배경)
  - 심볼 + 라벨 + 설명 표시

- 데이터 컬럼
  - 키워드
  - PC 검색량
  - 모바일 검색량
  - 합계 (강조)
  - 문서 수 / 상품 수 / 파워링크 (타입별)
  - 경쟁 비율 (색상 코딩)

#### 6. 결과 요약
- 판단별 개수 표시
  - 🔥 우수: X개
  - ✅ 양호: X개
  - ⚠️ 경고: X개
  - ❌ 위험: X개
- 총 분석 완료 개수

#### 7. 판단 기준 카드
- 4개 카드 (가로 배치)
- 각 판단 레벨별 기준값 표시
- 설명 포함

## 🎯 주요 기능

### ✅ 구현 완료
1. **키워드 입력**
   - 여러 줄 입력
   - 실시간 개수 표시
   - 최대 100개 제한

2. **검색 타입 전환**
   - 탭으로 타입 선택
   - 타입별 컬럼 변경

3. **실시간 사용량 표시**
   - 숫자 표시
   - 프로그레스 바
   - 색상 변경 (70%, 90% 기준)

4. **키워드 분석**
   - 로딩 상태
   - 에러 처리
   - 결과 표시

5. **판단 아이콘 강조**
   - 심볼 크게 표시
   - 색상 배경
   - 라벨 + 설명

6. **엑셀 다운로드**
   - CSV 파일 생성
   - UTF-8 BOM
   - 판단 포함

7. **결과 요약**
   - 판단별 개수
   - 통계 표시

8. **반응형 디자인**
   - 테이블 스크롤
   - 모바일 대응

## 🎨 디자인 특징

### 색상 스키마
- **우수 (🔥)**: 빨간색 (#DC2626)
- **양호 (✅)**: 녹색 (#16A34A)
- **경고 (⚠️)**: 노란색 (#CA8A04)
- **위험 (❌)**: 회색 (#4B5563)
- **Primary**: Indigo (#4F46E5)

### 타이포그래피
- 헤더: 2xl, bold
- 테이블 헤더: xs, uppercase
- 데이터: base, 숫자는 우측 정렬
- 판단 라벨: sm, bold

### 레이아웃
- 최대 너비: 7xl (1280px)
- 패딩: 1.5rem
- 카드: 둥근 모서리, 그림자
- 간격: 일관된 spacing

## 📊 사용 예시

### 1. 키워드 입력
```
네이버 블로그
키워드 분석
마케팅 도구
```

### 2. 분석 결과
| 판단 | 키워드 | PC | 모바일 | 합계 | 문서수 | 비율 |
|------|--------|-----|--------|------|--------|------|
| 🔥 우수 | 네이버 블로그 | 15,000 | 25,000 | 40,000 | 18,000 | 0.45 |
| ✅ 양호 | 키워드 분석 | 8,000 | 12,000 | 20,000 | 18,000 | 0.90 |
| ⚠️ 경고 | 마케팅 도구 | 10,000 | 15,000 | 25,000 | 40,000 | 1.60 |

```
일일 사용량: 3 / 20,000
```

## 🔧 기술 스택

- **TypeScript** - 타입 안전성

- **Axios** - HTTP 클라이언트
- **Vite** - 빌드 도구
